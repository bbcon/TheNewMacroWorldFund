How to wire this up:
  1. Generate CSVs: data/outputs/performance/trades/<trade_id>_daily.csv and .../<trade_id>_summary.csv.
  2. Copy them into docs/data/outputs/performance/trades/ (so GitHub Pages can serve them).
  3. Replace the sample data below with fetch calls (see inline comment in JS).
  4. Push to main; enable Pages on the repo using the /docs folder.
